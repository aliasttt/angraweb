# راهنمای نصب و راه‌اندازی پروژه Django

این پروژه یک وبسایت داینامیک برای شرکت برنامه‌نویسی و طراحی وب است که با Django ساخته شده است.

## ویژگی‌ها

✅ سیستم احراز هویت (ورود، ثبت‌نام، خروج)
✅ پنل ادمین کامل برای مدیریت همه چیز
✅ صفحات جداگانه برای خدمات مختلف:
   - طراحی وبسایت
   - طراحی اپلیکیشن موبایل
   - فروشگاه اینترنتی
   - سئو و بهینه‌سازی
   - هاستینگ و دامنه
   - طراحی UI/UX
✅ فرم‌های تماس و درخواست قیمت با ذخیره در دیتابیس
✅ مدیریت پروژه‌ها، پکیج‌ها، خدمات، گواهینامه‌ها
✅ سیستم وبلاگ
✅ نظرات مشتریان
✅ چندزبانه (ترکی، انگلیسی، فارسی، عربی)

## نصب و راه‌اندازی

### 1. نصب وابستگی‌ها

```bash
pip install django pillow
```

### 2. اجرای Migrations

```bash
python manage.py makemigrations
python manage.py migrate
```

### 3. ایجاد کاربر ادمین

```bash
python manage.py createsuperuser
```

### 4. اجرای سرور توسعه

```bash
python manage.py runserver
```

سپس به آدرس `http://127.0.0.1:8000` بروید.

## دسترسی به پنل ادمین

پس از ایجاد کاربر ادمین، به آدرس زیر بروید:
```
http://127.0.0.1:8000/admin/
```

## ساختار پروژه

```
angraweb/
├── angraweb_project/     # تنظیمات اصلی پروژه
├── main/                 # اپلیکیشن اصلی
│   ├── models.py         # مدل‌های دیتابیس
│   ├── views.py          # ویوها
│   ├── urls.py           # URLها
│   ├── forms.py          # فرم‌ها
│   └── admin.py          # پنل ادمین
├── templates/            # قالب‌های HTML
│   ├── base.html         # قالب پایه
│   └── main/             # قالب‌های صفحات
├── static/               # فایل‌های استاتیک
│   ├── css/
│   ├── js/
│   └── ...
└── media/                # فایل‌های آپلود شده
```

## مدل‌های دیتابیس

### Service (خدمات)
- مدیریت خدمات مختلف
- پشتیبانی چندزبانه
- آیکون و تصویر

### Package (پکیج‌ها)
- پکیج‌های مختلف با قیمت
- ویژگی‌های هر پکیج

### Project (پروژه‌ها)
- نمایش پروژه‌های انجام شده
- تصاویر و لینک‌ها

### ContactMessage (پیام‌های تماس)
- ذخیره پیام‌های فرم تماس
- مدیریت وضعیت پیام‌ها

### QuoteRequest (درخواست قیمت)
- درخواست‌های قیمت از مشتریان
- مدیریت وضعیت درخواست‌ها

### BlogPost (پست‌های وبلاگ)
- سیستم وبلاگ
- پشتیبانی چندزبانه

### Certificate (گواهینامه‌ها)
- نمایش گواهینامه‌ها
- تصویر و PDF

### Testimonial (نظرات مشتریان)
- نظرات و امتیازدهی مشتریان

## استفاده از پنل ادمین

1. وارد پنل ادمین شوید
2. می‌توانید:
   - خدمات جدید اضافه کنید
   - پکیج‌ها را مدیریت کنید
   - پروژه‌ها را اضافه/ویرایش کنید
   - پیام‌های تماس را مشاهده کنید
   - درخواست‌های قیمت را مدیریت کنید
   - پست‌های وبلاگ بنویسید
   - و خیلی بیشتر...

## صفحات موجود

- `/` - صفحه اصلی
- `/services/` - لیست خدمات
- `/service/<slug>/` - جزئیات یک خدمت
- `/web-design/` - صفحه طراحی وبسایت
- `/mobile-app/` - صفحه طراحی اپلیکیشن موبایل
- `/ecommerce/` - صفحه فروشگاه اینترنتی
- `/seo/` - صفحه سئو
- `/hosting/` - صفحه هاستینگ
- `/ui-ux/` - صفحه طراحی UI/UX
- `/packages/` - لیست پکیج‌ها
- `/projects/` - لیست پروژه‌ها
- `/project/<slug>/` - جزئیات پروژه
- `/about/` - درباره ما
- `/resume/` - رزومه و گواهینامه‌ها
- `/contact/` - تماس با ما
- `/quote/` - درخواست قیمت
- `/blog/` - وبلاگ
- `/blog/<slug>/` - جزئیات پست وبلاگ
- `/login/` - ورود
- `/register/` - ثبت‌نام
- `/logout/` - خروج
- `/lang/<lang_code>/` - تغییر زبان

## نکات مهم

1. فایل‌های استاتیک (CSS, JS, تصاویر) در پوشه `static/` قرار دارند
2. فایل‌های آپلود شده در پوشه `media/` ذخیره می‌شوند
3. برای تولید، باید `collectstatic` را اجرا کنید:
   ```bash
   python manage.py collectstatic
   ```
